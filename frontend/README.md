# Pipeline Builder

It is workflow application use to provide flow between nodes and also ensure the flow is DAG or not.Cool UI and robust feture are in this application.

# Frontend (React)

- React
- ReactFlow
- zustand (state management)
- tailwindcss

# Backend (Python)

- Fastapi
- uvicorn

## Features

- Node abstraction using BaseNode
- modern styled node types
- Dynamic TextNode with variable detection
- Undo / Redo / Reset functionality
- Modern UI with Tailwind
- Backend integration via API
- Analysis modal with DAG status
